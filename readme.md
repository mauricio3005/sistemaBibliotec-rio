# Contexto
A criação do projeto tem como motivação a criação de um sistema spring para treinar meus conhecimentos
# Overview
A intenção aqui é criar um sistema de bibliotecas via Spring que faça o gerenciamento do banco de dados pelos endpoints.
O sistema tem como modelo principal o livro, que esse será emprestado aos usuários daquela biblioteca.
Outros modelos como autor e enums serão também incluídos para melhorar o desenvolvimento.
# Stack
- Java 
- Spring
